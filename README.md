 LangGraph Streaming Chatbot (Ollama + Streamlit)

A real-time streaming chatbot built using LangGraph, Ollama (Llama 3.1) and Streamlit.
This bot supports token-by-token streaming, conversation memory, and a clean chat UI.

> Features:

 .Real-time streaming responses (live typing effect)

. LangGraph state-based memory

. Ollama (Llama 3.1) local LLM

. Streamlit chat UI

. Fast & fully local (no OpenAI API required)

. Clean separation of frontend & backend
